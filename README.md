# System Requirements 
- python 3.8+
- anaconda or miniconda 

# Getting started 
- create new conda environment 
```bash
conda create -n space-robotics python=3.8 -y
conda activate space-robotics
```
- install dependencies 
```bash
pip install -r requirements.txt
```

# Example
```bash
cd lecture1
python spacecraft_sim.py
```
